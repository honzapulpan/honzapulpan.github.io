Polynom 3. stupně¶

  • $at^3+bt^2+ct+d$
  • $d=0$
  • $a=1$
  • výsledný polynom: $t^3+bt^2-(1+b)t$

a,b,c,t = symbols('a b c t', real=True)
raw_poly = t**3 + b*t**2 - (1+b)*t
raw_poly
Out[50]:
$$b t^{2} + t^{3} - t \left(b + 1\right)$$

Vyjádření Rayleighova podílu pomocí dosazení obecného polynomu za neznámou funkci.

diff_poly = diff(raw_poly, t)

Najdeme stacionární bod a určíme, zda se jedná o minimum.

In [60]:
diff_rq = [simplify(diff(rq, x)) for x in [b,c]]
res = solve(diff_rq,(b,c))

diff2_rq = [[simplify(rq.diff(x).diff(y)) for x in [b,c]] for y in [b,c]]
diff2_rq

sol = []
for i in range(0, len(res)):
    uxx = simplify(diff2_rq[0][0].subs([(b,res[i][0]),(c,res[i][1])]))
    uxy = simplify(diff2_rq[0][1].subs([(b,res[i][0]),(c,res[i][1])]))
    uyy = simplify(diff2_rq[1][1].subs([(b,res[i][0]),(c,res[i][1])]))
    hess = N(uxx*uyy - uxy**2)
    if hess > 0:
        if N(uxx) > 0:
            sol.append(res[i])
sol
Out[60]:
$$\left [ \left ( -2, \quad - \frac{\sqrt{133}}{21} + \frac{2}{3}\right )\right ]$$
